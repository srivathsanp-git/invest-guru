# Invest Guru

A React + TypeScript investment analysis dashboard with mobile-first UX.

## Features included
- Asset performance: 1m, 3m, 1y, 5y, lifetime
- S&P 500 screener with 52-week/MA filters
- Financial metrics with hover tooltips
- Insider trades + analyst sentiment
- Responsive UI, dark professional theme

## Run locally
1. npm install
2. npm run dev
